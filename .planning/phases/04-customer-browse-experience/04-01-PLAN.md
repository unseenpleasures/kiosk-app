  - `_topSpacer` = `div.scroll-spacer.scroll-spacer--top` with `height: 0px`
  - `_gridContainer` = `div.catalogue-grid`
  - `_bottomSpacer` = `div.scroll-spacer.scroll-spacer--bottom`
- Create sentinels: `_topSentinel` = `div.scroll-sentinel.scroll-sentinel--top`, `_bottomSentinel` = `div.scroll-sentinel.scroll-sentinel--bottom`
- Place `_topSentinel` before `_topSpacer`, `_bottomSentinel` after `_bottomSpacer`
- Append the scroll container to screen, append screen to app
- Call `renderGridWindow()` to render the initial batch
- After initial batch is rendered, use `requestAnimationFrame` to measure `_rowHeight` from the first card tile's `offsetHeight`, then set up the IntersectionObserver

**`renderGridWindow()` function:**
- Clear `_gridContainer.innerHTML = ''`
- Calculate the end index: `var end = Math.min(_windowStart + _windowSize, _filtered.length)`
- Loop from `_windowStart` to `end`, calling `createCardTile(_filtered[i])` and appending to `_gridContainer`
- Update spacer heights: `_topSpacer.style.height = calculateSpacerHeight(_windowStart) + 'px'` and `_bottomSpacer.style.height = calculateSpacerHeight(_filtered.length - end) + 'px'`

**`calculateSpacerHeight(itemCount)` function:**
- If `_rowHeight === 0` return 0
- Calculate: `var rows = Math.ceil(itemCount / 5); return rows * _rowHeight;`

**`setupObserver()` function:**
- If `_observer` exists, disconnect it first
- Create `new IntersectionObserver(onSentinelIntersect, { root: scrollContainer, rootMargin: '200px 0px' })` where `scrollContainer` is the `.catalogue-scroll` element (the scrolling parent)
- Observe both `_topSentinel` and `_bottomSentinel`

**`onSentinelIntersect(entries)` function:**
- Wrap body in `requestAnimationFrame` to batch DOM mutations
- For each entry where `entry.isIntersecting`:
  - If `entry.target === _bottomSentinel`: call `appendNextBatch()`
  - If `entry.target === _topSentinel`: call `prependPrevBatch()`

**`appendNextBatch()` function:**
- If `_windowStart + _windowSize >= _filtered.length`, return (no more cards)
- Remove first `_batchSize` children from `_gridContainer`
- Advance `_windowStart` by `_batchSize`
- Calculate new end: `var end = Math.min(_windowStart + _windowSize, _filtered.length)`
- Append new cards from `_windowStart + _windowSize - _batchSize` to `end`
- Update both spacer heights

**`prependPrevBatch()` function:**
- If `_windowStart === 0`, return (already at top)
- Calculate new start: `var newStart = Math.max(0, _windowStart - _batchSize)`
- Remove last `(_windowStart - newStart)` children from `_gridContainer`
- Prepend cards from `newStart` to `_windowStart` (use `insertBefore` with `_gridContainer.firstChild`)
- Set `_windowStart = newStart`
- Update both spacer heights

**`resetVirtualScroll()` function:**
- Set `_windowStart = 0`
- If `_gridContainer` exists, set its parent's (`.catalogue-scroll`) `scrollTop = 0`

**`resetCatalogueState()` function** (will be called from app.js home button in Plan 02):
- Set `_activeCategory = null`, `_searchQuery = ''`
- Clear the search input value if it exists: `var searchInput = document.getElementById('catalogue-search'); if (searchInput) { searchInput.value = ''; }`
- Deselect all category chips: find all `.chip.chip--active` elements, remove `chip--active` class
- Copy `_products.slice()` into `_filtered`
- Call `resetVirtualScroll()` then `renderGridWindow()`

function onSearchInput(event) {
  clearTimeout(_searchDebounceTimer);

function applyFilters() {
  var query = _searchQuery;

  _filtered = _products.filter(function(p) {

    emptyMsg.className = 'catalogue-empty';
    emptyMsg.textContent = query ? 'No cards found for "' + query + '"' : 'No cards in this category';

  }
}
