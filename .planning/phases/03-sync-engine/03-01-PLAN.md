<objective>
Create the Shopify sync engine module (sync.js) that fetches all products via cursor-based GraphQL pagination, upserts product metadata into IndexedDB, caches product images into Cache API, and checkpoints the cursor for failure recovery.

Purpose: This is the data pipeline that populates the kiosk with product data before events. Without it, the app shows "Sync Required" permanently. The sync engine must be robust against network failures (cursor checkpoint) and memory-safe on A9X hardware (sequential image caching).

Output: src/sync.js — a standalone module exporting syncAll(onProgress) that Plan 02 (admin panel) will wire to the UI.
</objective>

**GraphQL query string (var, not const):**

```javascript
var PRODUCTS_QUERY = 'query ($cursor: String) { products(first: 250, after: $cursor) { nodes { id title handle productType tags createdAt updatedAt featuredImage { url altText width height } } pageInfo { hasNextPage endCursor } } }';
```

Note: Use `featuredImage.url` (NOT `src` or `originalSrc` — deprecated per RESEARCH.md).

**function fetchProductPage(cursor):**
- POST to `https://${SHOPIFY_STORE_DOMAIN}/api/${API_VERSION}/graphql.json`
- Headers: `Content-Type: application/json`, `X-Shopify-Storefront-Access-Token: ${STOREFRONT_TOKEN}`
- Body: `JSON.stringify({ query: PRODUCTS_QUERY, variables: { cursor: cursor || null } })`
- Check `res.ok`, throw `Error('HTTP ' + res.status)` on failure
- Return `res.json()`
- Use string concatenation for URL (no template literals — ES2017 safe)

**function cacheImagesSequential(urls, errors):**
- Takes array of image URL strings and an errors array (mutated)
- Uses `urls.reduce(function(chain, url) { ... }, Promise.resolve())` pattern for sequential execution
- For each URL: `caches.open(SYNC_CACHE_NAME)` then `fetch(url)` then `cache.put(url, response)` if `response.ok`
- On catch: push `{ url: url, error: err.message }` to errors array — NEVER abort the chain
- This prevents memory pressure on A9X from parallel image fetches

**function syncAll(onProgress):**
- Module-level variables: `var cursor = null; var pageNum = 0; var totalProducts = 0; var newProducts = 0; var errors = [];`
- Inner recursive function `fetchNext()`:
  1. Call `fetchProductPage(cursor)`
  2. Extract `data.data.products.nodes` and `data.data.products.pageInfo`
  3. Increment `pageNum`
  4. For each product node, check if existing via `dbGet('products', product.id)`:
     - If not existing, increment `newProducts`
     - Increment `totalProducts`
     - Call `dbPut('products', { id: product.id, title: product.title, handle: product.handle, category: product.productType, tags: product.tags, imageUrl: product.featuredImage ? product.featuredImage.url : null, imageAlt: product.featuredImage ? product.featuredImage.altText : null, createdAt: product.createdAt, updatedAt: product.updatedAt })`
  5. After all upserts resolve (Promise.all), call `cacheImagesSequential()` with the page's image URLs (filter null featuredImage)
  6. Checkpoint cursor: `dbPut('sync_meta', { key: 'currentCursor', value: pageInfo.endCursor })`
  7. Also checkpoint page progress: `dbPut('sync_meta', { key: 'pagesComplete', value: pageNum })`
  8. Fire `onProgress({ page: pageNum, products: totalProducts, newProducts: newProducts })` if callback provided
  9. If `pageInfo.hasNextPage === true`, recurse to `fetchNext()`
  10. If no more pages (sync complete):
      - Write `dbPut('sync_meta', { key: 'lastSyncAt', value: new Date().toISOString() })`
      - Write `dbPut('sync_meta', { key: 'productCount', value: totalProducts })`
      - Clear cursor checkpoint: `dbPut('sync_meta', { key: 'currentCursor', value: null })`
      - Clear pages checkpoint: `dbPut('sync_meta', { key: 'pagesComplete', value: null })`
      - Return stats object: `{ total: totalProducts, newProducts: newProducts, errors: errors }`

- Before calling `fetchNext()`, check for interrupted sync:
  ```javascript
  return dbGet('sync_meta', 'currentCursor').then(function(saved) {
    cursor = (saved && saved.value) ? saved.value : null;
    return dbGet('sync_meta', 'pagesComplete');
  }).then(function(savedPages) {
    pageNum = (savedPages && savedPages.value) ? savedPages.value : 0;
    return fetchNext();
  });
  ```

- Wrap the entire syncAll body in a try/catch-style `.catch()` that:
  - Does NOT clear the cursor checkpoint (preserves resume point)
  - Returns `{ total: totalProducts, newProducts: newProducts, errors: errors, aborted: true, abortReason: err.message }`
