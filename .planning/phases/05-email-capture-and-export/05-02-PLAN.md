**Function: `escapeCsvField(value)`** — add near the bottom of admin.js, before the `loadAndRenderSyncStatus` function:
```javascript
function escapeCsvField(value) {
  if (value.indexOf(',') !== -1 || value.indexOf('"') !== -1 || value.indexOf('\n') !== -1) {
    return '"' + value.replace(/"/g, '""') + '"';
  }
  return value;
}
```

**Function: `buildCsvFilename(eventName, eventDate)`** — add after `escapeCsvField`:
```javascript
function buildCsvFilename(eventName, eventDate) {
  var safeName = eventName.toLowerCase().replace(/\s+/g, '-').replace(/[^a-z0-9\-]/g, '');
  return 'emails-' + safeName + '-' + eventDate + '.csv';
}
```
