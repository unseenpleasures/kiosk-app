<objective>
Convert all absolute root paths to relative paths across the PWA codebase and prepare the repository for GitHub Pages deployment.

Purpose: GitHub Pages serves project pages from a subdirectory URL (https://unseenpleasures.github.io/kiosk-app/). Absolute paths like `/src/app.js` resolve to `unseenpleasures.github.io/src/app.js` (wrong). Relative paths like `./src/app.js` resolve to `unseenpleasures.github.io/kiosk-app/src/app.js` (correct). This plan converts all ~35 path strings and bumps the SW cache version.

Output: All 5 source files updated with relative paths, .gitignore created, ready for git push.
</objective>
